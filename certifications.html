Certifications




Voici les certifications que j'ai choisies..

La certification Pix

La certification Pix est un outil développé par le ministère de l'Éducation nationale pour évaluer les compétences numériques. Elle permet d'évaluer vos compétences dans différents domaines tels que la recherche d'informations, la protection de la vie privée et la création de contenu numérique. Elle est souvent utilisée par les recruteurs pour évaluer les compétences numériques des candidats.


La certification HTML5/CSS3

La certification HTML5/CSS3 est une certification qui prouve vos compétences dans le développement web. Elle m'est particulièrement utile en tant qu'étudiant en BTS SIO qui souhaite travailler dans ce domaine. Les compétences en HTML5/CSS3 sont essentielles pour créer des sites web modernes et adaptatifs. La certification AWS (Amazon Web Services) est une certification qui prouve votre expertise dans le cloud computing. La certification AWS prouve que vous avez les compétences nécessaires pour travailler avec les différents services AWS et les intégrer dans des solutions cloud pour les entreprises.



La certification AWS

AWS est une plate-forme de cloud computing proposée par Amazon qui offre une large gamme de services pour les entreprises et les organisations.
